# ValenbiciAPI26T8

Proyecto Maven Java que conecta con una base de datos MariaDB en AWS (Amazon RDS).

## Descripción
Aplicación que obtiene datos de las estaciones Valenbici desde la API del geoportal de Valencia y los almacena en una base de datos en la nube (AWS).

## Tecnologías
- Java 17
- Maven
- MySQL Connector J 9.2.0
- AWS RDS (MariaDB)

## Configuración
- Host: AWS RDS endpoint
- Puerto: 3306
- Base de datos: valenbicibd
